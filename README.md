##	Audio Spectogram - OpenGL, Python, PyAudio



##	Setup
>	Install Python
>	Install Pip
>	Install CMake
1.	Create * Activate Environment:
	-	`python -m venv .venv`
	-	`.venv\Scripts\activate.bat`
2.	Install Dependencies:
	-	`pip install -r requirements.txt`
3.	Run (in root directory):
	-	`make`
